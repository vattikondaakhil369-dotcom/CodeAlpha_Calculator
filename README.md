# 🧮 CodeAlpha Calculator

Welcome to the **CodeAlpha Calculator** project!

This web-based calculator is developed using **HTML, CSS, and JavaScript**, providing a smooth, interactive, and user-friendly experience for performing everyday mathematical calculations. The calculator features a clean and modern interface, making it easy to use across desktops, tablets, and mobile devices.

It supports all essential arithmetic operations, including addition, subtraction, multiplication, and division, while also offering keyboard input support for enhanced usability and faster calculations.

This project demonstrates fundamental frontend development concepts such as DOM manipulation, event handling, responsive web design, and user interface development. Special attention has been given to creating an attractive layout and ensuring a seamless user experience.
