 CodeAlpha Calculator--
Welcome to the CodeAlpha Calculator project! This web based calculator is developed using HTML, CSS, and JavaScript, providing a smooth and interactive experience for performing everyday mathematical calculations.
The calculator is designed with a clean and modern user interface, making it easy to use across different devices and screen sizes. It supports all basic arithmetic operations, including addition, subtraction, multiplication, and division, while also offering keyboard input support for enhanced usability.
This project demonstrates fundamental frontend development concepts such as DOM manipulation, event handling, responsive web design, and user interface development. Special attention has been given to creating a visually appealing layout and ensuring a seamless user experience.

 Key Highlights

- Perform basic arithmetic calculations instantly
- Responsive design for desktop, tablet, and mobile devices
- Keyboard support for faster input
- Real-time display updates
- Clean and modern user interface
- Built using pure HTML, CSS, and JavaScript

 Purpose of the Project--
This calculator was developed as part of the **CodeAlpha Frontend Development Internship Program**. The objective of the project was to strengthen practical frontend development skills by building a fully functional and user-friendly web application from scratch.
The project reflects my understanding of web technologies and my ability to create interactive applications using modern frontend development practices.
